<h2><span class="mw-headline" id="Pr.C3.A9sentation_rapide_:_comment_ins.C3.A9rer_des_coordonn.C3.A9es_facilement">Présentation rapide&#160;: comment insérer des coordonnées facilement</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mod%C3%A8le:Coord/documentation&amp;action=edit&amp;section=1" title="Modifier la section : Présentation rapide : comment insérer des coordonnées facilement">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Ce modèle sert à insérer des <a href="/w/index.php?title=Coordonn%C3%A9es_g%C3%A9ographiques&amp;action=edit&amp;redlink=1" class="new" title="Coordonnées géographiques (page inexistante)">coordonnées géographiques</a> dans un article&#160;; il insère en même temps un lien vers l’outil «&#160;GeoHack&#160;», qui donne accès à de nombreux outils de vues satellites, cartes,&#160;etc. comme <span class="lang-en" lang="en"><a href="/wiki/Google_Maps" title="Google Maps">Google Maps</a></span> par exemple.
</p><p>Pour insérer des coordonnées dans un article, suivez la procédure suivante (le manuel complet est plus bas)&#160;:
</p>
<ol><li> Trouvez les coordonnées de l’endroit, avec un logiciel (comme <span class="lang-en" lang="en"><a href="/w/index.php?title=Google_Earth&amp;action=edit&amp;redlink=1" class="new" title="Google Earth (page inexistante)">Google Earth</a></span>) ou un site adapté (comme <a rel="nofollow" class="external text" href="http://tools.freeside.sk/geolocator/geolocator.html">GeoLocator</a>, <a rel="nofollow" class="external text" href="http://mapper.acme.com">ACME Mapper</a>, <a href="/wiki/G%C3%A9oportail" title="Géoportail">Géoportail</a>…), ou d’après une source digne de foi. Vous pouvez les trouver dans trois formats, en général&#160;: soit sous la forme «&#160;26.456, -3.765&#160;» (degrés décimaux ou <b>dec</b>), soit «&#160;26°24.5′ N, 3°58.25′ W&#160;» (degrés et minutes puis fractions de minutes, ou <b>DM</b>), soit «&#160;26°24′30″ N 3°58′15″&#160;» (degrés minutes et secondes, <b>DMS</b>).</li>
<li> Écrivez à l’endroit souhaité de l’article le code suivant&#160;: <code>{{coord|<i>vos coordonnées</i>}}</code>, en séparant les coordonnées par une barre verticale. En format décimal, cela donne&#160;: <code>{{coord|26.456|-3.765}}</code>&#160;; en format DM, ceci&#160;: <code>{{coord|26|24.5|N|3|58.25|W}}</code>&#160;; en format DMS, ceci&#160;: <code>{{coord|26|24|30|N|3|58|15|W}}</code>. 
<ul><li> <u>Note</u>&#160;: pour DM et DMS, n’oubliez pas de préciser N/S/E/W.</li></ul></li>
<li> Ajoutez éventuellement des paramètres après une autre barre verticale. Par exemple, pour une ville, ajoutez <code>type:city</code>, pour un bâtiment <code>type:landmark</code>&#160;; vous pouvez préciser l’échelle avec <code>scale:20000</code> (échelle 1:<a href="/w/index.php?title=Mod%C3%A8le:Nombre&amp;action=edit&amp;redlink=1" class="new" title="Modèle:Nombre (page inexistante)">Modèle:Nombre</a>), la région du monde avec <code>region:FR</code> (FR étant le code <a href="/w/index.php?title=ISO_3166-1&amp;action=edit&amp;redlink=1" class="new" title="ISO 3166-1 (page inexistante)">ISO 3166-1 alpha 2</a> de la France). Pour une ville française vue à l’échelle 1:<a href="/w/index.php?title=Mod%C3%A8le:Nombre&amp;action=edit&amp;redlink=1" class="new" title="Modèle:Nombre (page inexistante)">Modèle:Nombre</a>, vous écrirez donc&#160;: <code>{{coord|26|24|30|N|3|58|15|W|type:city_region:FR_scale:10000}}</code>.
<ul><li> <u>Note</u>&#160;: ces paramètres sont séparés par un tiret bas&#160;; liste complète <b><a href="/w/index.php?title=Mod%C3%A8le:GeoTemplate/Utilisation&amp;action=edit&amp;redlink=1" class="new" title="Modèle:GeoTemplate/Utilisation (page inexistante)">sur cette page</a></b>.</li></ul></li>
<li> Par défaut, les coordonnées sont écrites là où le modèle est placé&#160;; vous pouvez les afficher dans le titre en ajoutant <code>|display=title</code>. Vous pouvez aussi choisir d’afficher des coordonnées décimales en format DMS en ajoutant <code>|format=dms</code>, ou des coordonnées DM/DMS en format décimal avec <code>format=dec</code>. Exemple&#160;: <code>{{coord|26.456|-3.765|format=dms}}</code></li>
<li> Enfin, n’oubliez pas de <a href="/w/index.php?title=Aide:Sourcer&amp;action=edit&amp;redlink=1" class="new" title="Aide:Sourcer (page inexistante)">préciser la source</a> des coordonnées, par exemple avec une <a href="/w/index.php?title=Aide:Notes&amp;action=edit&amp;redlink=1" class="new" title="Aide:Notes (page inexistante)">note</a>. Ainsi, en cas de contestation, on sait comment retrouver l’origine de votre choix.</li></ol>
<p>Le modèle complet peut donc ressembler à ceci&#160;: 
</p>
<dl><dd> <code>{{coord|26|24|30|N|3|58|15|W|type:city_region:FR_scale:10000|format=dec|display=title}}</code>. </dd></dl>
<p>En cas de doute sur les paramètres, mieux vaut ne pas recopier le modèle d’une autre page mais se contenter de ne mettre que les coordonnées, sans paramètres&#160;: ça marchera très bien ainsi. Le reste de cette page explique le fonctionnement en détail du modèle <a href="/wiki/Mod%C3%A8le:Coord" title="Modèle:Coord">{{coord}}</a>.
</p>

<h2><span class="mw-headline" id="Utilisation">Utilisation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mod%C3%A8le:Coord/documentation&amp;action=edit&amp;section=2" title="Modifier la section : Utilisation">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li> Ce modèle <a href="/wiki/Mod%C3%A8le:Coord" title="Modèle:Coord">{{Coord}}</a> permet de créer un lien vers <a rel="nofollow" class="external text" href="https://toolserver.org/~geohack/geohack.php?language=fr">Geohack</a>, un outil du toolserver qui donne une liste de liens sources vers des cartes, des vues satellites, des photos<a href="/w/index.php?title=Mod%C3%A8le:Etc.&amp;action=edit&amp;redlink=1" class="new" title="Modèle:Etc. (page inexistante)">Modèle:Etc.</a> ce pour une position géographique donnée par sa <a href="/wiki/Latitude" title="Latitude">latitude</a>, sa <a href="/wiki/Longitude" title="Longitude">longitude</a>, et son <a href="/wiki/%C3%89chelle_(proportion)" class="mw-redirect" title="Échelle (proportion)">échelle)</a> de visualisation. </li>
<li> Ce modèle permet également de standardiser la notation des <a href="/w/index.php?title=Coordonn%C3%A9es_g%C3%A9ographiques&amp;action=edit&amp;redlink=1" class="new" title="Coordonnées géographiques (page inexistante)">coordonnées géographiques</a>. Il utilise principalement des coordonnées terrestres (il est possible de préciser <code>globe:moon</code> ou <code>globe:mars</code>, sans réel effet pour l’instant) de type <i><a href="/w/index.php?title=WGS84&amp;action=edit&amp;redlink=1" class="new" title="WGS84 (page inexistante)">WGS84</a></i> (<a href="/wiki/Latitude" title="Latitude">latitude</a> - <a href="/wiki/Longitude" title="Longitude">longitude</a>). Ces coordonnées peuvent être <b>entrées</b> et <b>affichées</b> de manière indépendante dans&#160;:
<ul><li> le <a href="/wiki/Syst%C3%A8me_d%C3%A9cimal" title="Système décimal">système décimal</a> avec des degrés décimaux (DD), la précision dépend alors du nombre de chiffres rentrés après la virgule.   </li>
<li> le <a href="/w/index.php?title=Syst%C3%A8me_sexag%C3%A9simal&amp;action=edit&amp;redlink=1" class="new" title="Système sexagésimal (page inexistante)">système sexagésimal</a> ou degrés-minutes-secondes (DMS), la précision dépend alors du fait que l’on rentre les coordonnées jusqu’au secondes (DMS), ou que jusqu’au minutes (DM seulement), voir que jusqu’au degrés (D seulement).</li></ul></li>
<li> Le <b><a href="/w/index.php?title=Wikip%C3%A9dia:WikiMiniAtlas&amp;action=edit&amp;redlink=1" class="new" title="Wikipédia:WikiMiniAtlas (page inexistante)">WikiMiniAtlas</a></b> est activé, donc un petit globe <a href="/wiki/Fichier:Geographylogo.svg" class="image"><img alt="Geographylogo.svg" src="https://upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Geographylogo.svg/18px-Geographylogo.svg.png" width="18" height="18" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Geographylogo.svg/27px-Geographylogo.svg.png 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Geographylogo.svg/36px-Geographylogo.svg.png 2x" data-file-width="160" data-file-height="160" /></a> s’affiche juste avant les coordonnées et montre un atlas interactif&#160;: cliquez pour afficher une carte configurable.</li>
<li> Ces coordonnées géographiques sont lisibles et exportables en <a href="/w/index.php?title=Geo_(microformat)&amp;action=edit&amp;redlink=1" class="new" title="Geo (microformat) (page inexistante)">Géoformat</a>&#160;: voir <a href="//en.vikidia.org/wiki/WP:GEOUF" class="extiw" title="en:WP:GEOUF">en:WP:GEOUF</a> et <a rel="nofollow" class="external free" href="http://microformats.org/wiki/geo">http://microformats.org/wiki/geo</a> <span style="cursor:help;font-family:monospace;font-weight:bold;font-size:small" title="Langue&#160;: anglais">(en)</span>.</li>
<li> <a href="/wiki/Mod%C3%A8le:Coord" title="Modèle:Coord">{{Coord}}</a> est également utilisé par des outils qui extraient les coordonnées depuis la base de données, comme <a href="/w/index.php?title=Google_Earth&amp;action=edit&amp;redlink=1" class="new" title="Google Earth (page inexistante)">Google Earth</a> ou le <a href="/w/index.php?title=Wikip%C3%A9dia:WikiMiniAtlas&amp;action=edit&amp;redlink=1" class="new" title="Wikipédia:WikiMiniAtlas (page inexistante)">WikiMiniAtlas</a>. </li></ul>
<h2><span class="mw-headline" id="Syntaxe">Syntaxe</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mod%C3%A8le:Coord/documentation&amp;action=edit&amp;section=3" title="Modifier la section : Syntaxe">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Voici quatre exemples de syntaxe possible&#160;:
</p>
<ul><li> <tt>{{coord|<i>latitude</i>|<i>longitude</i>|<i>paramètres des coordonnées</i>|<i>paramètres du modèle</i>}}</tt></li>
<li> <tt>{{coord|<i>lat deg</i>|<i>N/S</i>|<i>lon deg</i>|<i>E/W</i>|<i>paramètres des coordonnées</i>|<i>paramètres du modèle</i>}}</tt></li>
<li> <tt>{{coord|<i>lat deg</i>|<i>lat min</i>|<i>N/S</i>|<i>lon deg</i>|<i>lon min</i>|<i>E/W</i>|<i>paramètres des coordonnées</i>|<i>paramètres du modèle</i>}}</tt></li>
<li> <tt>{{coord|<i>lat deg</i>|<i>lat min</i>|<i>lat sec</i>|<i>N/S</i>|<i>lon deg</i>|<i>lon min</i>|<i>lon sec</i>|<i>E/W</i>|<i>paramètres des coordonnées</i>|<i>paramètres du modèle</i>}}</tt></li></ul>
<h3><span class="mw-headline" id="Param.C3.A8tres_du_mod.C3.A8le">Paramètres du modèle</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mod%C3%A8le:Coord/documentation&amp;action=edit&amp;section=4" title="Modifier la section : Paramètres du modèle">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li> Les <i><a href="/w/index.php?title=Coordonn%C3%A9es_g%C3%A9ographiques&amp;action=edit&amp;redlink=1" class="new" title="Coordonnées géographiques (page inexistante)">coordonnées géographiques</a></i>, celles-ci peuvent être écrites au <a href="/wiki/Syst%C3%A8me_d%C3%A9cimal" title="Système décimal">format décimal</a> (auquel cas «&#160;latitude&#160;» et «&#160;longitude&#160;» prennent leurs valeurs décimales, comme par exemple «&#160;<code>-45.76843</code>&#160;» ou «&#160;<code>87.234567</code>&#160;» (utilisez un point comme séparateur décimal), ou au <a href="/w/index.php?title=Syst%C3%A8me_sexag%C3%A9simal&amp;action=edit&amp;redlink=1" class="new" title="Système sexagésimal (page inexistante)">format sexagésimal</a>&#160;: <i>deg</i> pour les degrés, <i>min</i> pour les minutes, <i>sec</i> pour les secondes avec «&#160;<code>N</code>&#160;» ou «&#160;<code>S</code>&#160;» pour la latitude et «&#160;<code>E</code>&#160;» ou «&#160;<code>W</code>&#160;» pour la longitude (n’utilisez pas la lettre «&#160;<code>O</code>&#160;»&#160;!)</li>
<li> <i>Les paramètres de coordonnées</i>, ce sont les paramètres d’<a href="/wiki/%C3%89chelle_(proportion)" class="mw-redirect" title="Échelle (proportion)">échelle</a> qui pourront être compris par <a rel="nofollow" class="external text" href="https://toolserver.org/~geohack/geohack.php?pagename=Wikip%C3%A9dia:WikiMiniAtlas&amp;language=fr">Geohack</a>, comme «&#160;<code>type:</code>&#160;», «&#160;<code>scale:</code>&#160;», «&#160;<code>region:</code>&#160;», «&#160;<code>altitude:</code>&#160;»… Si il y en a plusieurs en même temps, il faut les séparer par un tiret bas «&#160;_&#160;» (exemple&#160;: «&#160;<code>type:landmark_scale:50000</code>&#160;»). Pour la liste exhaustive voir <b><a href="/w/index.php?title=Mod%C3%A8le:GeoTemplate/Utilisation&amp;action=edit&amp;redlink=1" class="new" title="Modèle:GeoTemplate/Utilisation (page inexistante)">GeoTemplate/Utilisation</a></b>.</li>
<li> <i>Les paramètres du modèle</i>, ce sont «&#160;<code>display</code>&#160;», «&#160;<code>format</code>&#160;» et «&#160;<code>name</code>&#160;»&#160;:
<ul><li> <code>display</code> peut prendre les valeurs <code>inline</code>, ou <code>title</code>, ou encore <code>inline,title</code>. La syntaxe <code>display=inline</code> (par défaut) permet d’afficher les coordonnées dans le texte, là où le modèle est placé&#160;; <code>display=title</code> affiche les coordonnées juste en dessous de la barre de titre, à droite&#160;; <code>display=inline,title</code> fait les deux à la fois. Note&#160;: utilisez <code>title</code> pour signifier que les coordonnées en question concernent tout l’article.</li>
<li> <code>format</code> force le format d’affichage si on le veut différent du format des coordonnées géographiques entrées. <code>format=dec</code> donne un affichage décimal, <code>format=dms</code> un affichage sexagésimal, et <code>format=dms long</code> un affichage sexagésimal en français sur deux lignes (pratique dans les <a href="/w/index.php?title=Aide:Infobox&amp;action=edit&amp;redlink=1" class="new" title="Aide:Infobox (page inexistante)">infobox</a>, mais déconseillé dans les titres, utilisez alors <code>format=dms</code>). Par défaut, l’affichage est dans le même format que les arguments.</li>
<li> <code>name</code> sert à annoter les coordonnées pour les services externes. S’il est activé, le nom spécifié remplace le nom de l’article sur les cartes générées automatiquement. Par défaut, le nom est le titre de la page, il n’y a donc pas besoin de le spécifier pour les coordonnées dans les <a href="/w/index.php?title=Aide:Infobox&amp;action=edit&amp;redlink=1" class="new" title="Aide:Infobox (page inexistante)">infobox</a> ou dans le titre. Il est utile dans le cas où l’on souhaite utiliser dans un article le modèle «&#160;Coord&#160;» pour situer un point géographique mais qui n’est pas celui de l’article en question.</li></ul></li></ul>
<h3><span class="mw-headline" id="Erreurs_courantes">Erreurs courantes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mod%C3%A8le:Coord/documentation&amp;action=edit&amp;section=5" title="Modifier la section : Erreurs courantes">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Les erreurs courantes suivantes sont à éviter, ou le modèle ne fonctionnera pas correctement&#160;:
</p>
<ul><li> N’utilisez pas de virgule comme séparateur décimal, mais uniquement un point.</li>
<li> Ne mettez pas d’espace entre le signe négatif (-) et les nombres&#160;: -2.876 est correct, - 2.876 ne l’est pas.</li>
<li> N’écrivez pas «&#160;O&#160;» pour Ouest mais «&#160;W&#160;».</li></ul>

<h3><span class="mw-headline" id="Liste_exhaustive_des_possibilit.C3.A9s">Liste exhaustive des possibilités</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mod%C3%A8le:Coord/documentation&amp;action=edit&amp;section=8" title="Modifier la section : Liste exhaustive des possibilités">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li> Si les <a href="/w/index.php?title=Coordonn%C3%A9es_g%C3%A9ographiques&amp;action=edit&amp;redlink=1" class="new" title="Coordonnées géographiques (page inexistante)">coordonnées géographiques</a> entrées sont dans le <a href="/wiki/Syst%C3%A8me_d%C3%A9cimal" title="Système décimal">système décimal</a>, voir <b><a href="/w/index.php?title=Mod%C3%A8le:Coord/Exemple_dec&amp;action=edit&amp;redlink=1" class="new" title="Modèle:Coord/Exemple dec (page inexistante)">Coord/Exemple en décimal</a></b></li>
<li> Si les <a href="/w/index.php?title=Coordonn%C3%A9es_g%C3%A9ographiques&amp;action=edit&amp;redlink=1" class="new" title="Coordonnées géographiques (page inexistante)">coordonnées géographiques</a> entrées sont dans le <a href="/w/index.php?title=Syst%C3%A8me_sexag%C3%A9simal&amp;action=edit&amp;redlink=1" class="new" title="Système sexagésimal (page inexistante)">système sexagésimal</a>, voir <b><a href="/w/index.php?title=Mod%C3%A8le:Coord/Exemple_dms&amp;action=edit&amp;redlink=1" class="new" title="Modèle:Coord/Exemple dms (page inexistante)">Coord/Exemple en sexagésimal</a></b></li></ul>
<h2><span class="mw-headline" id="Personnalisation_de_l.27affichage">Personnalisation de l'affichage</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mod%C3%A8le:Coord/documentation&amp;action=edit&amp;section=9" title="Modifier la section : Personnalisation de l'affichage">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Par défaut, les coordonnées s’affichent dans le format dans lequel elles sont données. Il vous est possible de changer ce comportement.
</p>
<ul><li> Pour toujours afficher les coordonnées au format DMS, ajoutez le code suivant à <a href="/wiki/Sp%C3%A9cial:Ma_page/vector.css" title="Spécial:Ma page/vector.css">votre vector.css</a>&#160;:
<dl><dd> <code>.geo-default { display: inline } .geo-nondefault { display: inline } <br />.geo-dec { display: none }   .geo-dms { display: inline }</code></dd></dl></li>
<li> Pour toujours afficher les coordonnées au format décimal, ajoutez le code suivant à <a href="/wiki/Sp%C3%A9cial:Ma_page/vector.css" title="Spécial:Ma page/vector.css">votre vector.css</a>&#160;:
<dl><dd> <code>.geo-default { display: inline } .geo-nondefault { display: inline }<br />.geo-dec { display: inline } .geo-dms { display: none }</code></dd></dl></li>
<li> Pour afficher les deux formats ensemble, ajoutez le code suivant à <a href="/wiki/Sp%C3%A9cial:Ma_page/vector.css" title="Spécial:Ma page/vector.css">votre vector.css</a>&#160;:
<dl><dd> <code>.geo-default { display: inline } .geo-nondefault { display: inline }<br />.geo-dec { display: inline }   .geo-dms { display: inline }<br />.geo-multi-punct { display: inline }</code></dd></dl></li></ul>
<h2><span class="mw-headline" id="Remarque_importante">Remarque importante</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mod%C3%A8le:Coord/documentation&amp;action=edit&amp;section=10" title="Modifier la section : Remarque importante">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li> Avant toute modification du modèle présent, notez que les noms de classe «&#160;<code>geo</code>&#160;», «&#160;<code>latitude</code>&#160;» et «&#160;<code>longitude</code>&#160;» sont utilisés pour générer le <a href="/w/index.php?title=Microformat&amp;action=edit&amp;redlink=1" class="new" title="Microformat (page inexistante)">microformat</a> avec <a href="/w/index.php?title=Geo_(microformat)&amp;action=edit&amp;redlink=1" class="new" title="Geo (microformat) (page inexistante)">Géoformat</a> (voir <a rel="nofollow" class="external text" href="http://microformats.org/wiki/microformats">microformats.org</a>) et NE DOIVENT PAS ÊTRE MODIFIÉS.</li></ul>
