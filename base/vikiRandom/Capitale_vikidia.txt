<p>Une <b>capitale</b> est une <a href="/wiki/Ville" title="Ville">ville</a> qui occupe le premier rang d'un <a href="/wiki/%C3%89tat" title="État">pays</a>, d'une <a href="/wiki/Province" title="Province">province</a> ou d'une partie de pays sur un certain plan, notamment <a href="/wiki/Politique" title="Politique">politique</a> ou <a href="/wiki/%C3%89conomie" title="Économie">économique</a>, parfois les deux (voir plus bas). 
</p><p>Généralement, un pays a une seule capitale politique, où siège notamment le <a href="/wiki/Gouvernement" title="Gouvernement">gouvernement</a>. Mais certains pays en ont deux, comme les <a href="/wiki/Pays-Bas" title="Pays-Bas">Pays-Bas</a>&#160;: <a href="/wiki/Amsterdam" title="Amsterdam">Amsterdam</a> en est la capitale constitutionnelle<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">1</a></sup> et <a href="/wiki/La_Haye" title="La Haye">La Haye</a> la capitale administrative (où siègent le gouvernement, le <a href="/wiki/Parlement" class="mw-disambig" title="Parlement">Parlement</a> et les <a href="/wiki/Ambassade" title="Ambassade">ambassades</a>).
</p><p>Par extension, on peut aussi parler de capitale pour une ville qui joue un rôle essentiel dans un secteur d'activité&#160;: capitale <a href="/wiki/Industrie" title="Industrie">industrielle</a>, <a href="/wiki/%C3%89conomie" title="Économie">économique</a>, <a href="/wiki/Culture" title="Culture">culturelle</a>, etc. Ainsi, par exemple au <a href="/wiki/Maroc" title="Maroc">Maroc</a>, <a href="/wiki/Rabat" title="Rabat">Rabat</a> est la capitale politique et <a href="/wiki/Casablanca" title="Casablanca">Casablanca</a> la capitale économique.
</p><p><img alt="Attention" src="//download.vikidia.org/vikidia/fr/images/thumb/3/3f/Attention.svg/20px-Attention.svg.png" title="Attention" width="20" height="19" srcset="//download.vikidia.org/vikidia/fr/images/thumb/3/3f/Attention.svg/30px-Attention.svg.png 1.5x, //download.vikidia.org/vikidia/fr/images/thumb/3/3f/Attention.svg/40px-Attention.svg.png 2x" data-file-width="237" data-file-height="220" /> La capitale n'est pas obligatoirement la ville la plus peuplée du pays. Ainsi <a href="/wiki/Washington_(ville)" class="mw-redirect" title="Washington (ville)">Washington</a> est bien moins peuplé que <a href="/wiki/New_York" title="New York">New York</a> ou <a href="/wiki/Los_Angeles" title="Los Angeles">Los Angeles</a> aux <a href="/wiki/%C3%89tats-Unis" title="États-Unis">États-Unis</a>. Il en est de même pour <a href="/wiki/Canberra" title="Canberra">Canberra</a> et de <a href="/wiki/Sydney" title="Sydney">Sydney</a> ou <a href="/wiki/Melbourne" title="Melbourne">Melbourne</a> en <a href="/wiki/Australie" title="Australie">Australie</a>; idem pour <a href="/wiki/Brasilia" title="Brasilia">Brasilia</a> et <a href="/wiki/Rio_de_Janeiro" title="Rio de Janeiro">Rio de Janeiro</a> ou <a href="/wiki/Sao_Paulo" class="mw-redirect" title="Sao Paulo">Sao Paulo</a> au <a href="/wiki/Br%C3%A9sil" title="Brésil">Brésil</a>.
</p>
