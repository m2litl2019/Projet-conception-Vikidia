			<h1 id="firstHeading" class="firstHeading" lang="fr">4J Studios</h1>

				<div id="mw-content-text" lang="fr" dir="ltr" class="mw-content-ltr"><div class="thumb tright"><div class="thumbinner" style="width:152px;"><a href="/wiki/Fichier:Logo_Studio_4J.jpg" class="image"><img alt="" src="//download.vikidia.org/vikidia/fr/images/5/5c/Logo_Studio_4J.jpg" width="150" height="104" class="thumbimage" data-file-width="150" data-file-height="104" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Fichier:Logo_Studio_4J.jpg" class="internal" title="Agrandir"></a></div>Le logo des studios 4J.</div></div></div>
<p><b>4J Studios</b> est un studio de développement de <a href="/wiki/Jeu-vid%C3%A9o" class="mw-redirect" title="Jeu-vidéo">jeux-vidéo</a> écossais basé à <a href="/w/index.php?title=Dundee&amp;action=edit&amp;redlink=1" class="new" title="Dundee (page inexistante)">Dundee</a> et à <a href="/w/index.php?title=Est_Linton&amp;action=edit&amp;redlink=1" class="new" title="Est Linton (page inexistante)">Est Linton</a>. La société travaille avec un large panel d'éditeurs tels que <a href="/w/index.php?title=ZeniMax_Media&amp;action=edit&amp;redlink=1" class="new" title="ZeniMax Media (page inexistante)">ZeniMax Media</a>, <a href="/wiki/Mojang" title="Mojang">Mojang</a> et <a href="/w/index.php?title=Microsoft_Studios&amp;action=edit&amp;redlink=1" class="new" title="Microsoft Studios (page inexistante)">Microsoft Studios</a>. Une majorité du personnel est issue de l'ancien studio <a href="/w/index.php?title=VIS_Entertainment&amp;action=edit&amp;redlink=1" class="new" title="VIS Entertainment (page inexistante)">VIS Entertainment</a>. 4J Studios ont également aidé à développer <a href="/wiki/Minecraft" title="Minecraft">Minecraft</a>. Depuis 2010, les studios travaillent sur les versions console de <a href="/wiki/Minecraft" title="Minecraft">Minecraft</a> avec <a href="/wiki/Sony" title="Sony">Sony</a>, <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> et <a href="/wiki/Mojang" title="Mojang">Mojang</a>. Ils ont reçu de nombreuses récompenses pour ses portages.
</p>
