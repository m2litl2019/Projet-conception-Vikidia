			<h1 id="firstHeading" class="firstHeading" lang="fr">Plante tinctoriale</h1>
									<div id="bodyContent" class="mw-body-content">
									<div id="siteSub">« Plante tinctoriale » expliqué aux enfants par Vikidia, l’encyclopédie junior</div>
								<div id="contentSub"></div>

				<div id="mw-content-text" lang="fr" dir="ltr" class="mw-content-ltr"><div class="thumb tright"><div class="thumbinner" style="width:202px;"><a href="/wiki/Fichier:Djerba_Henn%C3%A9_et_Harkous.jpg" class="image"><img alt="" src="//download.vikidia.org/vikidia/fr/images/1/15/Djerba_Henn%C3%A9_et_Harkous.jpg" width="200" height="150" class="thumbimage" data-file-width="200" data-file-height="150" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Fichier:Djerba_Henn%C3%A9_et_Harkous.jpg" class="internal" title="Agrandir"></a></div>Jeune mariée <a href="/wiki/Tunisie" title="Tunisie">tunisienne</a> dont les mains ont été tatouées avec une mixture à base de <a href="/wiki/Henn%C3%A9" title="Henné">henné</a></div></div></div>
<p>Une <b>plante tinctoriale</b> est une <a href="/wiki/Plante" class="mw-redirect" title="Plante">plante</a> d'où l'on peut extraire des <a href="/wiki/Colorant" title="Colorant">colorants</a>. Ce genre de plante est connu de l'humain depuis des milliers d'années, notamment au <a href="/wiki/Moyen-Orient" title="Moyen-Orient">Moyen-Orient</a>, en <a href="/wiki/Afrique_du_Nord" title="Afrique du Nord">Afrique du Nord</a> et en <a href="/wiki/Inde" title="Inde">Inde</a>. Son utilisation est encore présente dans ces pays.
</p>
<h2><span class="mw-headline" id="Utilisation_traditionnelle">Utilisation traditionnelle</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Plante_tinctoriale&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Modifier la section : Utilisation traditionnelle">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Plante_tinctoriale&amp;action=edit&amp;section=1" title="Modifier la section : Utilisation traditionnelle">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Dans de nombreux pays <a href="/wiki/Orient" class="mw-disambig" title="Orient">orientaux</a>, les femmes teignent leurs <a href="/wiki/Cheveux" class="mw-redirect" title="Cheveux">cheveux</a> et <a href="/wiki/Tatouage" title="Tatouage">tatouent</a> leurs <a href="/wiki/Main" title="Main">mains</a> et <a href="/wiki/Pied" title="Pied">pieds</a> avec des mixtures issues de plantes tinctoriales comme le <a href="/wiki/Henn%C3%A9" title="Henné">henné</a> et le <a href="/wiki/Safran" title="Safran">safran</a>.
</p>
<h2><span class="mw-headline" id="Quelques_plantes_tinctoriales">Quelques plantes tinctoriales</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Plante_tinctoriale&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Modifier la section : Quelques plantes tinctoriales">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Plante_tinctoriale&amp;action=edit&amp;section=2" title="Modifier la section : Quelques plantes tinctoriales">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li> La <a href="/wiki/Garance" title="Garance">garance</a>, qui donne un colorant <a href="/wiki/Rouge" title="Rouge">rouge</a>.</li>
<li> Le <a href="/wiki/Henn%C3%A9" title="Henné">henné</a>, qui donne une couleur allant de l'<a href="/wiki/Orange" class="mw-disambig" title="Orange">orange</a> au <a href="/wiki/Brun" title="Brun">brun</a></li>
<li> L'<a href="/w/index.php?title=Indigotier&amp;action=edit&amp;redlink=1" class="new" title="Indigotier (page inexistante)">indigotier</a> qui donne une couleur <a href="/wiki/Bleu_indigo" class="mw-redirect" title="Bleu indigo">bleu indigo</a>.</li>
<li> Le <a href="/wiki/Pastel" title="Pastel">pastel</a>, qui donne une couleur <a href="/wiki/Bleu" title="Bleu">bleu</a> clair.</li>
<li> Le <a href="/wiki/Safran" title="Safran">safran</a> ou <a href="/wiki/Crocus" title="Crocus">crocus</a>, qui teint en orange et fournit un <a href="/w/index.php?title=Aromate&amp;action=edit&amp;redlink=1" class="new" title="Aromate (page inexistante)">aromate</a>.</li>
<li> Le <a href="/w/index.php?title=Bois_de_br%C3%A9sil&amp;action=edit&amp;redlink=1" class="new" title="Bois de brésil (page inexistante)">bois de brésil</a>, arbre dont le bois sert à teindre en <a href="/wiki/Violet" title="Violet">violet</a>.</li>
<li> Le <a href="/wiki/Noyer" title="Noyer">noyer</a> et le <a href="/wiki/Ch%C3%A2taignier" title="Châtaignier">châtaignier</a> dont l'écorce ou le fruit servent à teindre en <a href="/wiki/Noir_(couleur)" class="mw-redirect" title="Noir (couleur)">noir</a></li></ul>
